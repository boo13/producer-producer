**Design direction:**
- "Duolingo for jobs" - friendly, encouraging, personality-driven
- Dark grey background (#1a1a1a or similar) - cards pop against it
- Light warm cards (cream/beige like v1) - approachable, not stark white
- Score badge as bold "sticker" - KEY design element, communicates job quality at a glance
- High scores (80+) = exciting, vibrant sticker
- Medium scores (60-79) = solid but muted
- Lower scores (<60) = subdued treatment
- Smooth animations throughout - satisfying micro-interactions

**v1 reference palette:**
- --window-bg: #f5f0e1 (warm cream)
- --title-bar: #d4cfc3 (muted beige)
- --ink: #2a2a2e (dark text)
- --accent: #e74c3c (warm red)

**Current v2 files:**
@v2/css/styles.css
@v2/index.html

<task type="auto">
  <name>Task 2: Create bold score sticker treatment</name>
  <files>v2/css/styles.css, v2/js/opportunities.js</files>
  <action>
Make the score badge a prominent "sticker" design element:

1. In CSS, redesign .job-card__score as a sticker:
   - Position: absolute, top-right corner of card (-8px, -8px offset to "stick out")
   - Size: min-width 56px, height 56px, border-radius 50% (circular sticker)
   - Font: bold VT323, large (1.5rem+)
   - Transform: rotate(-12deg) for playful angle
   - Box-shadow for "lifted" sticker effect

2. Create score tiers with distinct visual treatments:

   .job-card__score--high (80-100):
   - Background: linear-gradient(135deg, #2ecc71, #27ae60)
   - Color: white
   - Border: 3px solid white
   - Box-shadow: 0 4px 8px rgba(46,204,113,0.4)
   - Optional: subtle pulse animation on load

   .job-card__score--medium (60-79):
   - Background: var(--card-border) or #d4cfc3
   - Color: var(--ink)
   - Border: 2px solid var(--ink-light)
   - No special effects

   .job-card__score--low (<60):
   - Background: #e0e0e0 (grey)
   - Color: var(--ink-light)
   - Border: 1px dashed var(--ink-light)
   - Smaller size (48px), less prominent

3. In opportunities.js, update the card rendering to:
   - Add score tier class based on value
   - Ensure score displays as integer (no decimals)
   - Format: just the number, no "Score:" label (sticker speaks for itself)

4. Add CSS keyframe for high-score pulse:
   @keyframes score-pulse {
     0%, 100% { transform: rotate(-12deg) scale(1); }
     50% { transform: rotate(-12deg) scale(1.05); }
   }
   Apply with animation: score-pulse 2s ease-in-out infinite on --high

IMPORTANT: The score sticker should feel like a quality badge - high scores should make users excited to swipe right.
